# Changelog

All notable changes to this project are documented in this file.

## [0.2.0] - 2026-04-24

### Added

1. Optional API key authentication for protected trace endpoints via `AGENT_TRACER_API_KEY`.
2. In-memory write throttling via `AGENT_TRACER_RATE_LIMIT_PER_MINUTE`.
3. API tests for auth and rate-limit behavior.
4. Production baseline governance files and docs pack.
5. CI and release workflows for validation and tagged releases.
